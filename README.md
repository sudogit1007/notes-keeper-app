# Notes Keeper App

A simple web-based Notes Keeper application built using HTML, CSS, and JavaScript.  
It includes a basic login page and a dashboard where users can create, view, and manage notes.

---

## Features

- User login page (basic frontend authentication UI)
- Dashboard to manage notes
- Add new notes
- View saved notes
- Delete notes
- Clean and minimal UI

---

## Tech used

- HTML
- CSS
- JavaScript (Vanilla JS)
